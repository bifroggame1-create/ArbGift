# Competitor Research Report

## 1. THERMOS.GIFTS (Главный конкурент)

### Обзор
- **Создатель**: команда swap.coffee
- **Тип**: Telegram Mini App — агрегатор NFT-подарков
- **MAU**: 236.6K (август 2025), рост +315.2%
- **Языки**: EN, RU, UA, ZH

### Ключевые фичи
- **Агрегация цен**: сканирует MRKT, Tonnel, Portals — находит лучшую цену
- **Без комиссии для покупателя**: маркетплейсы-партнёры покрывают fees
- **Кэшбек**: при покупке через Thermos начисляется кэшбек в TON
- **Корзина**: можно добавлять несколько гифтов и покупать пачкой
- **Фильтры**: по цене, популярности, рарности

### UX/UI
- Чистый минималистичный интерфейс
- Карточки гифтов с ценой, рарностью, изображением
- Детальный просмотр: цена, рарность, продавец
- Кнопка "Buy" → подтверждение через кошелёк

### API (backend.thermos.gifts)
```
POST /api/v1/auth/api-token — аутентификация, JWT на 24ч
POST /api/v1/gifts/buy — покупка (до 20 гифтов, source: TONNEL/PORTALS/MRKT)
POST /api/v1/gifts/transfer — вывод гифтов
POST /api/v1/gifts/assign — внутренний трансфер (без комиссии)
POST /api/v1/gifts/sell — листинг (мин. 2 TON для Tonnel, 0.5 TON для Portals)
POST /api/v1/gifts/cancel-sale — отмена продажи
GET  /api/v1/gifts/me/on-sale — мои продажи
GET  /api/v1/balance/deposit — пополнение
POST /api/v1/balance/withdraw — вывод (fee 0.1 TON)
GET  /api/v1/tasks/{id} — статус операции
```

### Комиссии
- Покупка (себе): 0.1 TON
- Покупка+вывод (другим): 0.2 TON
- Вывод гифта: 0.1 TON
- Листинг/отмена: 0.1 TON за операцию
- Вывод баланса: 0.1 TON

### Известные проблемы
- Корзина сбрасывается при перезагрузке страницы (мобильный баг)
- Нет функции вывода TON (в разработке)

## 2. PORTALS.TG (portals / portalsmarket.co)

### Обзор
- **Основатель**: Roxman (крупнейший личный Telegram-канал)
- **Запуск**: май 2025
- **MAU**: 1.1M+
- **Доля рынка**: 50-60% от $160M рынка gift/NFT в Telegram
- **Одобрение**: Павел Дуров лично поддержал
- **Партнёры**: Snoop Dogg

### Навигация
1. **Store/Market** — активные листинги, фильтры, коллекции
2. **My Gifts** — инвентарь, листинг, отправка, вывод, бандлы
3. **Season** — задания, звёзды/поинты, лидерборд (топ-250), эксклюзивные награды
4. **Profile** — статистика, реферальная программа, баланс

### Торговля
- **Покупка**: мгновенная или оффер (0 или 7 дней)
- **Продажа**: одиночная, массовая, бандлы, пре-маркет
- **Офферы**: размещение, отмена, редактирование, коллекционные офферы
- **Фильтры**: коллекция, модель, бэкдроп, символ, цена, рарность

### Уникальные фичи
- **Zero commission** (первые 10 дней — 0% fees)
- **Pre-Market** — листинг до разлока
- **Mystery Portals** — сюрприз-паки с рандомными NFT
- **Реферальная программа**: 20-50% комиссия в TON
- **Season/Tasks**: геймификация для удержания
- **Ранние пользователи**: 0.1 TON за каждый уникальный NFT

### API (Python: `pip install portalsmp`)
```
search() — поиск гифтов с фильтрами
giftsFloors() — floor prices
filterFloors() — floor по атрибутам
collections() — метаданные коллекций
myPortalsGifts() — мои гифты
myPoints() — баланс поинтов
sale() / bulkList() — листинг
buy() — покупка (nft_id, owner_id, price)
makeOffer() / cancelOffer() / editOffer() — офферы
collectionOffer() — оффер на коллекцию
marketActivity() — глобальная активность
```

### Аутентификация
- Метод 1: из web.telegram.org network requests ("tma (auth)", 1-7 дней)
- Метод 2: Pyrogram/Kurigram (api_id + api_hash)

### Знаковые сделки
- $350,000 Porsche 911 GT3 за 18 Telegram "#1 Gifts" NFTs

## Выводы для нашего проекта

### Что делать как Thermos (наш главный конкурент):
1. **Агрегация цен** — ключевое УТП, сканировать все маркетплейсы
2. **Zero buyer fees** — маркетплейсы покрывают
3. **Кэшбек** — мотивация покупать через нас
4. **Чистый UI** — минимализм, карточки, фильтры
5. **Корзина** — batch purchasing

### Что взять у Portals (лидер рынка):
1. **Seasons/Tasks** — геймификация для retention
2. **Реферальная программа** с высокой комиссией (20-50%)
3. **Pre-Market** листинг
4. **Mystery boxes** (но с прозрачностью)
5. **Floor price** на каждую коллекцию
6. **Activity feed** для social proof

### Что взять у Marketapp:
1. **Rent** — уникальная фича аренды NFT
2. **Bundles** — пакетные предложения
3. **Веб-версия** — не только TMA
4. **Public API docs** — для интеграций
